# Study Dashboard

Study Dashboard is a web application built with ASP.NET Core MVC + Web API
**Technologies Used**  
Backend

- ASP.NET Core Web API (C#)
- Entity Framework Core
- JWT Authentication

Frontend

- ASP.NET Core MVC (Razor Pages)
- Bootstrap 5

Database

- Microsoft SQL Server
- Azure SQL Database

DevOps & Deployment

- Azure App Service
